# notes-main-bb

## 项目简介

`notes-main-bb` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为一组自包含的 HTML 文档集合，方便浏览与分发。

## 目录说明

```
notes-main-bb/
├── index.html          # 仓库根页面，提供导航或说明
├── pages/              # 存放主要 HTML 页面文件
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 静态资源（图片、样式、脚本等，如有需要）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：包含所有归档的 HTML 页面，每个文件均为独立可打开的完整文档。
- `assets/`：存放页面所需的公共静态资源，便于统一管理。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立归档，不依赖外部特定环境或服务。页面内容可能涵盖笔记、工具、演示或其他通用用途。所有页面均可在本地直接打开浏览，无需额外配置。

## 维护说明

- 新增页面：在 `pages/` 目录下创建新 HTML 文件，并确保其自包含（即所有资源引用均使用相对路径或内联方式）。
- 更新页面：直接修改对应 HTML 文件，保持文件结构清晰。
- 删除页面：移除不再需要的文件，并更新 `index.html` 中的导航链接（如有）。
- 提交规范：建议每次提交仅包含一个页面的新增或修改，并附上简要说明。
